Japan Trip Planner
Project for the course Software Development Environments  
Authors: Juan Luis Gavira and Adrián Hermida

Project Objective
This assignment consists of developing a simple software project with the assistance of generative AI and managing the entire workflow using Git exclusively through the command line.
The goals include:

Using AI as a collaborator during development

Applying a complete Git workflow (init, commits, branches, merges, remote)

Publishing the project through GitHub Pages

Documenting the entire process professionally

Project Description
Japan Trip Planner is an interactive web application that allows users to explore an illustrated map of Japan and view travel itineraries for several major cities.

The application includes:

A visual map of Japan

Buttons for cities such as Tokyo, Kyoto, Osaka, Hiroshima, and Nara

Dynamic itineraries displayed using JavaScript

Each itinerary contains:

Overview of the city

Main attractions

Local cuisine

Transportation information

Best time to visit

The content updates automatically when the user selects a city.

Use of AI in Development
Generative AI tools were used to support the project in the following ways:

Creating and expanding the itinerary content

Assisting with JavaScript logic and DOM manipulation

Debugging and improving HTML, CSS, and JavaScript

Adding the new city “Nara” as part of the feature-update branch

Refining the structure and clarity of the code
