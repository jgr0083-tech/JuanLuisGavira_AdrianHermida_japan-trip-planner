## **1. Project Objective**

This assignment focuses on developing a simple software project with the assistance of generative AI and managing the entire workflow using Git exclusively through the command line.

| **Goal** | **Description** |
|---------|------------------|
| **AI‑assisted development** | Use generative AI as a collaborator during the project. |
| **Git workflow mastery** | Apply a complete Git workflow: `init`, `add`, `commit`, `branch`, `merge`, `push`. |
| **Deployment** | Publish the project using **GitHub Pages**. |
| **Professional documentation** | Record the entire process clearly and formally. |

---

## **2. Project Description**

**Japan Trip Planner** is an interactive web application that allows users to explore a map of Japan and view itineraries for major cities.

| **Feature** | **Details** |
|-------------|-------------|
| **Interactive map** | Visual map of Japan with clickable regions. |
| **City selection** | Tokyo, Kyoto, Osaka, Hiroshima, and **Nara**. |
| **Dynamic itineraries** | Content updates automatically using JavaScript. |
| **Information included** | Attractions, food, transport, best time to visit, and city overview. |
